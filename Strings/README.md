# Strings

Solve string matching and manipulation problems with character and substring techniques.

## When to use it

- Substring search
- Sliding windows over strings
- Character frequency tracking

## Recognition patterns

- Use KMP for pattern matching
- Use last-seen indexes for sliding windows
- Normalize strings before comparison

## Typical complexity

O(n) or O(n + m) for most string problems.

## Problems in this folder

- 1. 003 Longest Substring Without Repeating Characters
- 2. 028 Find the Index of the First Occurrence in a String

## Interview tips

- Keep track of character positions and repeated patterns.
