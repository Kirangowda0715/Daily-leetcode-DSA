# BitManipulation

Use bitwise operations to solve number and parity problems with constant space.

## When to use it

- XOR cancellation
- Bit masks
- Isolate rightmost set bit
- Count set bits

## Recognition patterns

- Use XOR to cancel pairs
- Separate numbers by a differing bit
- Track counts with bit masks

## Typical complexity

O(n) time, O(1) space.

## Problems in this folder

- 1. 000 XOR Duplicate Number
- 2. 136 Single Number
- 3. 137 Single Number II
- 4. 260 Single Number III
- 5. 268 Missing Number

## Interview tips

- Use bit operations to cancel duplicates and extract masked values.
